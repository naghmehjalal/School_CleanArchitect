# school online system 

A simple website created with **ASP. Net Core 7**, **C#** and **MS SQL Server** allow members to download  courses online.


### Technologies
* Bootstrap 5
* ASP.NET Core 7
* C#
* MS SQL Server 2014, Entity Framework Core
* Web API
